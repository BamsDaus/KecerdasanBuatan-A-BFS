# Tugas Latihan untuk Algoritma A\* dan BFS (Breadth First Search)

### About this project

Pada project ini kedua algoritma diatas digambarkan menggunakan plugin p5.js untuk menggambarkan jalur yang ditempuh dari Kos menuju H5.

### Algoritma yang Digunakan

Algoritma A\* merupakan algoritma komputer yang menggunakan estimasi jarak dengan menggunakan pencarian jalur terdekat untuk mencapai tujuan dan memiliki fungsi heuristik yang digunakan sebagai dasar pertimbangan untuk menentukan pilihan sejumlah alternatif untuk mencapai sasaran dengan efektif.

Breadth First Search (BFS) adalah sebuah algoritma yang melakukan pencarian secara melebar yang mengunjungi simpul secara preorder yaitu mengunjungi suatu simpul kemudian mengunjungi semua simpul yang bertetangga dengan simpul tersebut terlebih dahulu.
